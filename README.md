# RefClubhouse

## Description

> The beginnings of a Django based application to Sports Official Associations to assist in managing their organizations. 

> RefClubhouse provides functionality for:
 - Managing membership rosters.
 - Provide the association with a default web site.
 - Communication with members through email, messaging and news items.
 - A document library.
 - An event calendar.
 - Scrimmage scheduling and assignment.
 - Surveys, polls and voting.
 - A photo gallery.
